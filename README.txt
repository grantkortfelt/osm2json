# osm2json

osm2json is a tool for converting OpenStreetMap (OSM) data to JSON format and vice versa.

## Features

- Convert OSM .poly data to .json GeoJSON
- Convert GeoJSON data to OSM .poly

## Installation

To install osm2json, clone the repository.

## Usage

Convert .poly to .json:
- Poly(file_path).convert()

Convert .json to .poly:
- Json(file_path).convert()

Use zealandia.json and zealandia.poly as example files
